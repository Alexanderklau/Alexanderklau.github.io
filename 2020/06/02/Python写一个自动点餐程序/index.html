为什么要写这个

公司现在用meican作为点餐渠道，每天规定的时间是早7：00-9：40点餐，有时候我经常容易忘记，或者是在地铁/公交上没办法点餐，所以总是没饭吃，只有去楼下711买点饭团之类的玩意儿，所以这是促使我写点餐小程序的原因。

点餐的流程

登录 —> 点餐 —> 提交

哈哈，是不是很简单，其实这个还好，说白了，就是登录上去，然后拿到cookie，保持一个登录状态，然后再去点餐，点餐就是构造请求，发送到指定的点餐URL上就可以了。
