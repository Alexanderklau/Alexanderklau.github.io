Elasticsearch for python API模块化封装

模块的具体功能

  1. 检测Elasticsearch节点是否畅通
  2. 查询Elasticsearch节点健康状态
  3. 查询包含的关键字的日志（展示前10条）
  4. 查询指定的索引下的数据，并且分页
  5. 输出所有日志(输出全部)
  6. 输出去重后的日志(分页，带关键字）
  7. 删除指定索引的值
  8. 往索引中添加数据
  9. 获取指定index、type、id对应的数据
  10. 更新指定index、type、id所对应的数据
  11. 批量插入数据
